# vcms
一、项目介绍  
     vcms原计划是做一个cms项目，目前只是完成了基础权限部分，后面有时间再完善，但是最主要的框架是没得问题的，项目中使用了springboot，springsecurity权限管理等知识点。

二、技术栈  
        后端：springboot+springsecurity+mybatis+kaptcha+swagger+generator等  
        前端：thymeleaf+jquery+layui+ztree等  
        数据库：mysql
